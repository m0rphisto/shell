So Ihr Lieben:
 Einfach eine PowerShell aufmachen, in das Verzeichnis quizgame reinhoppen
 und dann das Skript aufrufen. Die JSON Datei, bzw. die Quiz-Files müssen im
 Ordner quizzes gespeichert werden, da drinnen sucht das Skript.

 ---------------------------------
 Variante 1: Hilfe aufrufen

   PS> python quizgame.py -h

 ---------------------------------
 Variante 2: Quiz starten

   PS> python quizgame.py bsi

 ---------------------------------

Der Rest erklärt sich eigentlich von selbst.
Viel Spaß beim lernen.
